TTHK Timetable API (1.0.0)

This is the unofficial API to access TTHK's timetable data

Timetable

These are the main timetable endpoints

Get all timetables

Authorizations:
ApiKeyAuth

Responses

http://localhost:5637/api/timetable/

Response samples

  • 200
Content type
[
  • {
    • "tt_num": "",
    • "year": 0,
    • "text": "",
    • "datefrom": "yyyy-mm-dd",
    • "hidden": false
    }
]

Get all classes/groups related to the timetable

Authorizations:
ApiKeyAuth
path Parameters
timetableId
required
string
query Parameters
name
string

The name of the class to look for

color
string

The color of the class to look for

teachers
Array of arrays

An array of teacher ids that a class must have assigned to them, which is typically only one

classrooms
Array of arrays

An array of classroom ids that a class must have assigned to them

Responses

http://localhost:5637/api/timetable/{timetableId}/classes/

Get all lessons related to the timetable

Authorizations:
ApiKeyAuth
path Parameters
timetableId
required
string
query Parameters
subject
string

The id of the subject that the lesson should be associated with

classes
Array of arrays

An array of class ids that the lesson must have assigned to them

teachers
Array of arrays

An array of teacher ids that the lesson must have assigned to them

rooms
Array of arrays

An array of classroom ids that the lesson must have assigned to them

period_start
integer

Which period the lesson has to start at

period_end
integer

Which period the lesson has to end at

days
Array of arrays

An array of the names of days that the lesson is on

Responses

http://localhost:5637/api/timetable/{timetableId}/lessons/

Response samples

  • 200
Content type
[
  • {
    • "id": "*1",
    • "subject": "*1223",
    • "classes": [
      • "-467"
      ],
    • "teachers": [
      • "*140"
      ],
    • "rooms": [
      • "*29"
      ],
    • "periods": [
      • 7,
      • 7
      ],
    • "days": [
      • ""
      ]
    }
]

Timetable changes

These are the timetable changes endpoints, used to scrape changes from TTHK's website

Get all timetable changes

Authorizations:
ApiKeyAuth

Responses

http://localhost:5637/api/timetable_changes/

Response samples

  • 200
Content type
[
  • {
    • "dayLetter": "",
    • "date": { },
    • "class": "",
    • "period_span": [
      • 0,
      • 0
      ],
    • "teacher": "",
    • "info": ""
    }
]
