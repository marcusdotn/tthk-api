getAllLessons

path Parameters
timetableId
required
string
query Parameters
subject
string

The id of the subject that the lesson should be associated with

teachers
Array of any

An array of teacher ids that the lesson must have assigned to them

rooms
Array of any

An array of classroom ids that the lesson must have assigned to them

period_start
integer

Which period the lesson has to start at

period_end
integer

Which period the lesson has to end at

days
Array of any

An array of the names of days that the lesson is on
